# Holding AI Web Engine

Package-based SEO landing system built on Next.js App Router.

## Stack

- Next.js 16
- Tailwind CSS v4
- TypeScript (strict)
- Lucide React
- Radix UI Slot
- Framer Motion

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Included baseline

- Home page with package matrix and conversion triggers
- Dynamic package pages at `/packages/[slug]`
- Analytics event architecture (`dataLayer`, `gtag`, `amplitude` hooks)
- SEO baseline: metadata, OpenGraph, `robots.txt`, `sitemap.xml`, JSON-LD
- Dark-first visual language with glassmorphism and motion-safe behavior
