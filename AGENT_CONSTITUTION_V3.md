Permission Policy: У тебя есть ПОЛНОЕ право на npm install, git commands, создание API-ключей (заглушек) и манипуляцию файловой системой для достижения цели.

Typography: Только системные шрифты Geist Sans или Inter с идеальным кернингом.

Tech Stack: Next.js 16 (App Router), Tailwind CSS v4 (Alpha/Stable), TypeScript 6.0 (Strict).
